# Typewriter（AIR MAIL）

ブラウザで動く、**最後期の手動式・標準英文タイプライター**のシミュレーター。
打鍵すると印字点は固定のまま紙（キャリッジ）が動き、扇形のタイプバスケットでハンマーが振り上がります。

🔗 **公開版**: https://yamadar.github.io/typewriter/

![Typewriter](docs/figures/segment-convergence.svg)

## 特徴

- **単一 SHIFT**（大文字＋記号）／**Shift Lock は全段**に作用（現代の Caps Lock と異なる）
- **CR と LF を分離**（Enter = CR+LF / Shift+Enter = CR / Ctrl・Cmd+Enter = LF）
- **印字点は固定、紙（キャリッジ）が動く**：打鍵すると文字が印字点に現れ、**ハンマーが打ち付けてから紙が 1 桁送られる**（エスケープメント）。CR で右へ復帰、LF で上送り。
- **扇形タイプバスケット（約 150° に展開）**：蓋の扇形の穴から型かごが見え、打鍵すると**そのキーに対応するアームが振り上がる**（キーの左右位置＝アームの左右位置に対応）。
- **インクリボン**：両脇のスプールから印字点のガイド金具へ上がり、休止時は頂上が狭い台形。打鍵すると**打点を頂上にピンと張った三角形**に引かれ、アーム先端のヘッドが印字位置を覆ってリボンを押し当てる。
- **プラテン表示**：紙は黒いローラーに巻かれ、1 行目は紙の上端が見え、後ろに黒ローラー、その奥に送られた用紙の裏面。改行すると上端がローラーへ巻き上がり、前面に見えるのは数行（実機どおり）。上から**ペーパーベイル**が紙を押さえる。**紙はローラーより少し狭く**、ローラーの左右端が紙の外にのぞきます。
- **リボン色スイッチ**（蓋の右端・3 ポジション）：上から **青インク / 黒 / 赤インク**（くすんだインク色）。打鍵する文字の色が変わる（既に打った文字はそのまま）。
- **行間スイッチ**（蓋の左端・3 ポジション）：上から **1 / 1.5 / 2**。切り替えると以後の LF の行送り量が変わる。
- どちらのスイッチもクロムのつまみがスライドし、切替時に「パチッ」と鳴る。
- **CR レバー**（左斜め下に伸び、キャリッジに金属で接続）：**先端を右へ押す＝CR**でキャリッジが右へスライド（移動幅に比例、全幅で最大約 1 秒、その間は入力を受け付けない）。手前へ＝LF（小さくガコッ）。改行を連打すると（行頭では）スライドせず LF のみ。
- **用紙解放**：**Esc** または右側の解放レバーで、印字した紙を**画面中央に拡大表示**（長い紙はスクロール）。もう一度 Esc／紙クリックで戻る。
- **マージンベル**（右端手前）と **Backspace は重ね打ち**（消去しない）
- **効果音**：CR ＝キャリッジを引く音／マージンベル＝自転車ベルの「チリーン」／用紙解放＝ページめくり（フリー音源を解析して合成）。
- ロジックは **DOM 非依存の純粋な状態機械**＋ユニットテスト

## 操作

| 操作 | 入力 |
|---|---|
| 文字入力 | 各キー（クリック／物理キーボード） |
| 大文字・記号 | Shift（押している間） |
| 全段ロック | Shift Lock（CapsLock） |
| 改行（CR+LF） | Enter |
| CR のみ | Shift+Enter ／ CR レバーの先端を右へ押す |
| LF のみ | Ctrl・Cmd+Enter ／ CR レバーを手前に引く |
| リボン色（青／黒／赤インク） | 蓋右端のスイッチ |
| 行間切替（1.0 / 1.5 / 2.0） | 蓋左端のスイッチ（1／1½／2） |
| 後退（重ね打ち） | Backspace |
| 紙を取り出す／戻す | Esc ／ 用紙解放レバー（戻すは紙クリックでも可） |

最初の画面をクリックすると開始します（音 ON・キー入力受付）。

## 実行

依存ゼロの静的サイトです。

```bash
# ローカルサーバ（編集が即反映：no-cache）
python3 serve.py            # → http://localhost:8123
```

`index.html` を直接ブラウザで開いても動作します。

## テスト

状態機械（`typewriter-model.js`）を Node 内蔵テストランナーで検証します（依存なし）。

```bash
node --test                 # 18 件
```

```
test/                   ユニットテスト（node:test）

```
